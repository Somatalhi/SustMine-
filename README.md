# SustMine-
This repository contains the data, scripts, and supplementary materials associated with the paper "SustMine: A Framework for Integrating Sustainable Development Dimensions into Strategic Mine Planning," submitted to Mining, Metallurgy &amp; Exploration.
Citation
If you use SustMine in your research, please cite:
Altalhi, H., Awuah-Offei, K., Nicolosi, G., Al Moinee, A., and Al Habib, N. (2025). "SustMine: A Framework for Integrating Sustainable Development Dimensions into Strategic Mine Planning." Mining, Metallurgy & Exploration.
License
This repository is made available for academic and research use. Users are encouraged to modify and extend the framework for their own applications and to contribute improvements back to the repository.
